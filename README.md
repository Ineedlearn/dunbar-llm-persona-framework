# Social-Ring-Weighted LLM Agent Framework (SWLA)
### 社会年轮加权 LLM 智能体框架

> **Why give the convenience store owner the same compute budget as your partner?**  
> SWLA proposes differentiated computational resource allocation for LLM-based social simulation,  
> grounded in Dunbar's social brain hypothesis and Tamarit et al.'s (2018) cognitive resource constraint model.

---

## The Problem with Equal-Weight Deployment

Park et al. (2023) *Generative Agents* demonstrated 25 LLM agents spontaneously organizing parties,  
spreading rumors, and forming relationships — without being told to. Impressive.  
But it deploys every agent at equal compute weight.

This encodes a false assumption: **that every person in your social circle influences you equally.**  
Half a century of social psychology disagrees.
